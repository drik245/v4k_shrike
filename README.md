# v4k_shrike

Example projects for the Vicharak Shrike development boards — **Shrike Lite (RP2040)** and **Shrike Fi (ESP32-S3)** — featuring an on-board Renesas Forge SLG47910 FPGA.

## Project List

| # | Project | Board | Description |
|---|---------|-------|-------------|
| 1 | [Snake Game](snake_game/) | Shrike Lite + Shrike Fi | Classic snake game on SSD1306 OLED with joystick control |
| 2 | [Joystick Mouse](joystick_mouse/) | Shrike Fi | BLE HID mouse using a joystick — requires Bluetooth |
